# Yannled_Labo_React



# Init

`npx create-react-app my-app`

`npm init react-app my-app`

`yarn create react-app my-app`

# Run

`cd my-app`

`npm start`

# Test

`cd my-app`

`npm test`

# Build

`cd my-app`

`npm run build`
